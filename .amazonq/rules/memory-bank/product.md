# Product Overview

AI-powered project management app using Google Gemini AI for intelligent tracking, phase management, and team collaboration.

## Features
- Phase-based tracking with task management
- Interactive dashboard with real-time metrics
- Document management and viewer
- Team collaboration and workload distribution
- Revision control and testing integration
- My Work view and recently viewed projects
- Authentication and API key management
- **Agentic Task Execution**: 3-agent workflow (Doer, Tools, Tester) autonomously completes tasks
- **Agentic Change Management**: Automated change propagation across documents
- **Automated Milestones**: Dates calculated from task completion
- **Smart Notifications**: Email alerts with Phase 1 context when tasks ready

## Users & Use Cases
Project managers, dev teams, organizations needing AI-assisted multi-phase project tracking, automated reporting, document collaboration, autonomous task completion, and change impact analysis.
